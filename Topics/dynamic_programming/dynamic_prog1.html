Introduction

Dynamic Programming (DP) is a powerful problem-solving technique used in computer science to solve complex problems by breaking them down into simpler, overlapping subproblems. It is primarily an optimization over plain recursion, where previously computed results are reused to avoid redundant computations.

Dynamic Programming is commonly used for:

  1. Finding the optimal solution to a problem (e.g., minimum/maximum value, longest path, etc.)
  2. Counting the total number of possible solutions (e.g., number of ways to reach a target)

The best way to do DP is understanding the intuition behind it via multiple examples like :

Dice Problem (CSES)

Given a target sum n, you need to compute the total number of distinct sequences of dice rolls (each roll giving 1 to 6) that sum up to exactly n.

This is a classic Dynamic Programming problem where the recurrence relation is: 

f(n) = f(n-1) + f(n-2) + f(n-3) + f(n-4) + f(n-5) + f(n-6)
