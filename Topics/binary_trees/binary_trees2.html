Depth First Search

Preorder Traversal

Follows the Root → Left → Right order.

  • Visit the root node first
  • Traverse the left subtree
  • Traverse the right subtree
void preorder(Node* N) {
        if (N == nullptr) return;
    
        visit(N);             // Root
        preorder(N->left);    // Left
        preorder(N->right);   // Right
    }

Inorder Traversal

Follows the Left → Root → Right order.

  • Traverse the left subtree
  • Visit the root node
  • Traverse the right subtree
void inorder(Node* N) {
        if (N == nullptr) return;
    
        inorder(N->left);     // Left
        visit(N);             // Root
        inorder(N->right);    // Right
    }

Postorder Traversal

Follows the Left → Right → Root order.

  • Traverse the left subtree
  • Traverse the right subtree
  • Visit the root node
void postorder(Node* N) {
        if (N == nullptr) return;
    
        postorder(N->left);   // Left
        postorder(N->right);  // Right
        visit(N);             // Root
    }

Breadth First Search (BFS)

Also known as Level Order Traversal, BFS visits all nodes at each depth level before moving to the next level. It proceeds level by level, from top to bottom and left to right.

  • Starts from the root node
  • Uses a queue to keep track of nodes at the current level and their children, ensuring that all nodes of one level are processed before moving to the next
  • Processes nodes in FIFO (First-In-First-Out) order
void bfs(Node* root) {
          if (root == nullptr) return;
      
          std::queue<Node*> q;
          q.push(root);
      
          while (!q.empty()) {
              Node* current = q.front();
              q.pop();
      
              visit(current);  // Process the current node
      
              if (current->left)  q.push(current->left);
              if (current->right) q.push(current->right);
          }
      }
